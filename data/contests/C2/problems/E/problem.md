## Description

Given an integer n, determine the maximum possible GCD of two integers a and b such that 1 ≤ a < b ≤ n.

### Input

The first line of input contains a single integer $t$ ($1 \le t \le 2 \cdot 10^4$), the number of testcases.
The following $t$ lines each contain a single integer $n$ ($1 \le n \le 10^{18}$).

### Output

For each testcase, print a single integer — the maximum possible `gcd(a, b)` for $1 \le a < b \le n$.

## Constraints

- There will be $t$ testcases, where $1 \le t \le 2 \cdot 10^4$.
- For each testcase, there will be an integer $n$, where $1 \le n \le 10^{18}$.
